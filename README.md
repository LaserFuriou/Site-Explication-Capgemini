# Site-Explication-Capgemini

Site web vitrine présentant Capgemini à destination des candidats (stagiaires, alternants, jeunes diplômés). Il met en avant les valeurs de l'entreprise, ses engagements et permet d'envoyer une candidature.

## Pages

- **Accueil** (`index.html`) — Présentation générale, carousel de témoignages d'employés et lien vers le formulaire de candidature
- **Secteurs d'activités** (`SecteursActivités.html`) — Les domaines d'intervention de Capgemini
- **Durabilité environnementale** (`durabilite_environnementale.html`) — Les engagements écologiques de l'entreprise
- **Inclusion numérique** (`inclusion_numérique.html`) — Les actions pour réduire les inégalités numériques
- **Diversité et Inclusion** (`diversite_et_inclusion.html`) — La politique diversité de Capgemini
- **CV** (`cv.html` / `cv.php`) — Formulaire d'envoi de candidature

## Technologies

- HTML5
- CSS3
- JavaScript
- PHP (gestion du formulaire de candidature et du footer)

## Lancer le projet

Ce site nécessite un serveur PHP pour les fonctionnalités de formulaire. Vous pouvez l'exécuter localement avec :

```bash
php -S localhost:8000
```

Puis ouvrir [http://localhost:8000](http://localhost:8000) dans votre navigateur.

Pour les pages statiques uniquement (sans formulaire), il suffit d'ouvrir `index.html` directement dans un navigateur.
